  • html 5

    HTML é uma linguagem de marcação utilizada na construção de páginas na Web.

      Prós

    • Fácil de aprender e codificar.
    • Amplamente utilizado nos navegadores.
    • Navegação Móvel Facilitada.

      Contrás

    • Ele pode criar apenas páginas estáticas.
    • Não é ideal para o desenvolvimento de games.
    • Precisa escrever muito código para criar uma página da web simples.

  • css

    CSS é um mecanismo para adicionar estilo a um documento web.

      Prós

    • Fácil para o usuário personalizar a página online.
    • Redução do código HTML.
    • A estilização aplicada de forma consistente em vários sites.

      Contrás

    • Falta de padronização entre navegadores.
    • Trabalho extra para suportar diferentes navegadores.
    • Diferentes níveis de CSS.

  • javascript

    Linguagem de progamação de alto nível utilizada principalmente para aplicações web, sendo possível usar tanto como client-side e server-side.

    • Os menores erros impossibilitam a página de ser renderizada.

  • Nodejs

    Node.js é um software de código aberto, multiplataforma, baseado no interpretador V8 do Google e que permite a execução de códigos JavaScript fora de um navegador web.

      Prós

    • Multiplataforma.
    • Gerencioador de pacotes npm e uma comunidade ativa.

      Contrás

    • Não possui uma grande abrangência de funcionalidades próprias, forçando o desenvolvedor a utilizar bibiliotecas de terceiros.
    • O modelo de programação assíncrona do Node.js dificulta a manutenção do código.
    • Reduz o desempenho ao lidar com tarefas pesadas de computação.

  • Angular

    Angular é uma plataforma de aplicações web de código-fonte aberto e front-end baseado em TypeScript e utilizado com foco em aplicações web SPA.

      Prós

    • Reutilização de código com uso de componentes.
    • A natureza independente dos componentes simplifica os testes unitários.
    • Angular simplifica a maneira de gerenciar esses componentes. A arquitetura é construída de forma a ajudar o programador a localizar e desenvolver o código facilmente.

      Contrás

    • Opções limitadas de SEO e a baixa acessibilidade para rastreadores de mecanismos de pesquisa.
    • Devido a two-way data binding, especialmente em dispositivos antigos, a manipulação de dom angular também pode levar a uma compensação de desempenho, pois o navegador estará superocupado manipulando os elementos dom e consumirá muitos recursos de computação.
    • Aplicativos angulares podem ser bastante inchados devido ao excesso de abstração e recursos da estrutura que não podem ser descartados, pois são agrupados na estrutura.

      Empresas que utilizam

    • Console
    • Angular Prest

    • Allmyhomes
    • WeASSIST
    • Dxworks Site

  • Firebase

    O Firebase é um conjunto de serviços de hospedagem para qualquer tipo de aplicativo. Oferece hospedagem NoSQL e em tempo real de bancos de dados, conteúdo, autenticação social e notificações, ou serviços, como um servidor de comunicação em tempo real.

      Prós

    • Banco de dados confiável e extenso.
    • Hospedagem rápida e segura.
    • Autenticação do Firebase.

      Contrás

    • Falta de recursos de consulta. Você não pode consultar mais de uma chave por vez e não há método para filtrar seus dados. Todo o banco de dados é um arquivo JSON. Esse formato também elimina a possibilidade de modelar os dados.
    • Ele limita fortemente a migração de dados. Isso torna os usuários dependentes da plataforma e caso mude o conceito de base back-end, não há como transferir o app para outra fonte.
    • Suporte limitado para iOS em relação ao android.

      Empresas que utilizam

    • Alibaba Travels
    • Patreon
    • GitBook
    • Zé Delivery
    • Pier

  • Tailwindcss

    É um framework CSS visando a utilidade(utility-first), que fornece várias dessas classes utilitárias, opinativas e de propósito único.

      Prós

    • Reduz o tempo para estilizar uma página.
    • O Tailwind fornece a capacidade de criar temas responsivos para seus aplicativos da Web e remover todas as classes CSS não utilizadas. Com o PurgeCSS, o Tailwind ajuda você a manter seu CSS final o menor possível.
    • Altamente customizável.

      Contrás

    • Ele vai contra o princípio da “separação de interesses”.
    • Tailwind não possui muitos componentes de estilo significativos, como cabeçalhos, botões e barras de navegação.
    • Embora você esteja familiarizado com CSS, você precisa conhecer as classes predefinidas usadas no tailwindcss.

    • Whyp

    Karma permite executar código JavaScript em vários navegadores reais . Tem como principal objetivo tornar o desenvolvimento orientado a testes fácil, rápido e divertido.

  • Cypress

    Cypress é um framework de testes, código aberto e de fácil configuração. Focado em testes end-to-end.

      Prós

    • Cypress é rápido. Os testes são executados rapidamente e você pode ver facilmente o que aconteceu em cada etapa.
    • A interface é simples e intuitiva.
    • É flexível. Ele pode ser usado para uma variedade de tarefas diferentes, incluindo testes de end-to-end, testes de unidade e testes de integração.

      Contrás

    • O Cypress funciona apenas com Chrome, Firefox, Edge, Brave e Electron, portanto, se você precisar testar outros navegadores, precisará usar uma ferramenta diferente.
    • Atualmente, os teste são construídos em JavaScript. Se você estiver usando outro linguagem que o Cypress ainda não oferece suporte, talvez seja necessário procure uma alternativa.
    • Não é possível testar várias guias ou várias janelas do navegador ao mesmo tempo.

      Empresas que utilizam

    • Bitpanda
    • CircleCI
    • Alibaba Travels
    • Primer
    • Oxylabs
    • Revolut
